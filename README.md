# RadarChartToolkit

Ein Python-Toolkit zum Erstellen, Bearbeiten und Exportieren von Radar Charts.

## Features (geplant)

- JSON-Datenmodell
- Excel Import/Export
- Radar Plot Generator
- Interaktiver Editor
- Godot Integration

## Projektstruktur

- `models.py` – Datenmodell
- `json_handler.py` – JSON Import/Export
- `excel_handler.py` – Excel Import/Export
- `radar_plot.py` – Erzeugt Radar Charts
- `editor.py` – Später grafischer Editor
