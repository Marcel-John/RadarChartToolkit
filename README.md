# RadarChartToolkit

Ein Python-Toolkit zum Erstellen, Bearbeiten und Exportieren von Radar Charts.

## Installation

Clone the repository:

```bash
git clone https://github.com/Marcel-John/RadarChartToolkit.git
cd RadarChartToolkit
```

## Install dependencies

```bash
pip install -r requirements.txt
```

## (Recommended) Use a virtual environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
# source venv/bin/activate  # Mac/Linux
```

Then install dependencies:

```bash
python -m pip install -r requirements.txt
```

## Features (geplant)

- JSON-Datenmodell
- Excel Import/Export
- Radar Plot Generator
- Interaktiver Editor
- Godot Integration

## Projektstruktur

- `models.py` – Datenmodell
- `json_handler.py` – JSON Import/Export
- `excel_handler.py` – Excel Import/Export
- `radar_plot.py` – Erzeugt Radar Charts
- `editor.py` – Später grafischer Editor
